# fave_mobx

> 这是一个soundcloud的个人客户端 
## 使用到的技术
1. react
2. mobx 
3. typescript
4. react-router v4

mobx是一个mvvm的状态管理层,可以让你更多的关注ui层,
状态驱动能让你更关注compoennt的主要更新点.它值得你一试!这里是它的[中文文档](http://cn.mobx.js.org/)

## 功能
 - 主页,个人主页,播放,播放列表,评论列表,歌单列表,喜爱列表,
 - 图片的懒加载
 - 分页加载
 - 播放进度
 - 个人主页歌曲排序
